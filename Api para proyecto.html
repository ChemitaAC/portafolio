API WEB para validación de datos

Esta API se basa principalmente en el uso de dos propiedades las cuales son checkValidity() que sirve para revisar si los valores introducidos son validos, y setCustomValidity(), la cual delimita el rango de validez de un dato.

Y por medio de la condicional "if", es que se dara el rango en el que los valores se introduzcan y validen

Y es por medio de las propiedades de validez como lo son : customError, patternMismatch, rangeOverflow, rangeUnderflow, stepMismatch, tooLong, typeMismatch, valueMissing, valid

Esta api se utiliza para la validación de datos, como lo pueden ser datos enteros donde se pide que introduzcas un numero en un rango de valores especificos
